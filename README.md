# NLP Recommendation Chatbot
ユーザの入力から研究分野の興味を推定し、最適な研究室を推薦する対話型チャットボットです。  
単純な質問応答だけでなく、逆質問（フォローアップ質問）を行うことで、ユーザの関心を能動的に絞り込む仕組みを備えています。

## 概要
本プロジェクトは、ユーザが入力した文章を自然言語処理によって解析し、研究分野キーワードとの一致度をもとに研究室を推薦するシステムです。
特徴として、スコアが拮抗している場合に  システム側から選択式の逆質問を行い、興味を明確化する対話設計を採用しています。

## 主な特徴
### 1. 名詞ベースの興味推定（NLP）
Janome を用いて、ユーザ入力文から 名詞のみを抽出
抽出した名詞を、研究室ごとに定義された研究キーワード辞書と照合
一致度に応じてスコアを加算

| 判定方法 | スコア |
| 完全一致 |  3点  |
| 部分一致 |  2点  |

### 2. 研究室ごとのスコアリング
各研究室にスコアを保持
1回の入力につき、研究室ごとに最大スコアのみを採用
ノイズ的な単語の影響を抑え、興味の強さを安定して反映

### 3. 逆質問による興味の精緻化
一定確率で逆質問（フォローアップ質問）を行います。

例：
ちなみに、どちらの分野に興味がありますか？
機械学習
自然言語処理
どちらも興味ない

選択されたキーワードに関連する研究室のスコアを加算
「どちらも興味ない」を選択した場合は減点
ユーザの負担を抑えつつ、嗜好を対話的に明確化

### 4. 推薦判定ロジック
スコア上位1位と2位の差が一定以上になった場合、最終的なおすすめ研究室を提示
差が小さい場合は対話を継続

## 対話フロー
1. ユーザが興味分野を入力  
2. 名詞抽出・スコア更新
3. 必要に応じて逆質問を実施  
4. スコア差が十分になった時点で研究室を推薦  

※ 途中で 「列挙」 と入力すると、現在のスコア上位3研究室を確認できます。
